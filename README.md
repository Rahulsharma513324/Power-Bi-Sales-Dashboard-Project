Madhav Ecommerce Sales Dashboard


Project Overview

This project involves analyzing a comprehensive E-Commerce sales dataset to identify key performance indicators (KPIs) and business trends. The interactive Power BI dashboard provides actionable insights into sales volume, profit margins, and customer payment behaviors across different Indian states.


Key Insights & Features

Top Performance Metrics: Tracked Total Sales Amount (438K), Total Profit (37K), Total Quantity (5615), and Average Order Value (121K).

Category Analysis: Identified "Clothing" as the dominant category (63% of quantity), followed by Electronics (21%) and Furniture (17%).

Geographic Breakdown: Visualized sales distribution by state, highlighting Maharashtra and Madhya Pradesh as top-performing regions.

Payment Behavior: Analyzed customer payment preferences, revealing Cash on Delivery (COD) as the preferred mode (44%), followed by UPI and Debit Cards.

Monthly Trends: Monitored profit fluctuations month-over-month to spot seasonal peaks and dips.

Tools Used

Power BI: Used for data visualization, creating interactive slicers (Quarters, States), and drill-down analysis.

Data Cleaning: Processed raw sales data to ensure accuracy in state names and category classifications.

Data Modeling: Established relationships between sales tables and master data for accurate filtering.
